# 🐦 Space Flappy Network

A Java-based, object-oriented arcade game inspired by **Flappy Bird**, enhanced with a space-themed environment, real-time physics, procedural obstacle generation, and optional online asset loading.

Developed as part of the **Object-Oriented Programming through Java (OOPJ)** mini-project at **NMIMS MPSTME**.

---

## 📌 Project Overview

**Space Flappy Network** is a single-player arcade game built using **Java Swing and AWT**.  
The player controls a bird that must navigate through continuously spawning pipe obstacles while affected by gravity.

The game demonstrates core **Object-Oriented Programming principles**:
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

It also integrates:
- Event-driven programming (Swing Timer)
- Collision detection using `Rectangle`
- Online asset loading via HTTP
- Fallback rendering for offline mode

---

## 🎮 Features

- 🕹️ Simple one-key control (SPACE to flap)
- 🌌 Dynamic background (online or fallback gradient)
- 🧱 Procedural pipe generation
- ⚖️ Realistic gravity-based physics
- 💥 Collision detection system
- 🏆 Score tracking system
- ☠️ Game Over screen with restart option
- 🌐 Online image loading from GitHub (with fallback support)

---

## 🧠 Controls
